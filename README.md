# 🌸 Cutesy Room – Three.js Project

This is a small **cutesy room project** I created using **Three.js** and **Blender**.

It’s basically a **beginner-level project** where I explored how 3D scenes work on the web and got more comfortable with **JavaScript**, **Three.js**, and creating models in **Blender**.


##  About the Project

I made a cozy, aesthetic room in Blender and then brought it to life in the browser using Three.js.

The goal wasn’t to build something complex, but to:

* Understand how 3D models are used in web development
* Explore rendering, lighting, and camera in Three.js
* Practice integrating Blender models into a web scene
* Experiment and learn new things along the way

##  Tech Stack

* **JavaScript**
* **Three.js**
* **Blender**

## Features

* A cute, stylized 3D room
* Imported Blender models
* Basic lighting and scene setup
* Interactive camera (if you added controls)
